# cles-math-games

Interactive math games I built for my two kids to practice elementary school math. Lives at [cn814.github.io/cles-math-games](https://cn814.github.io/cles-math-games/) — runs entirely in the browser, no login required.

Built and iterated in response to what my kids were actually working on in class, so the games reflect real curriculum: number lines, counting strategies, place value, and arithmetic.

## Games

- **Count Back Stadium** — number line game for practicing subtraction by counting back, with step-by-step hop visualization
- **Number Line** — place numbers on a number line with progressively harder ranges
- **Number Comparison** — greater than / less than practice
- **Partial Sum** — break addition problems into place value chunks
- **Walking Game** — movement-based counting drill with pop-up answer checks

## Notes

Single-file HTML app — just `index.html`. No build step, no dependencies, no data collection. Open it in a browser and it works.
